# Unity_Facia_SDK
Easily bring the value of the expression captured by the TrueDepth camera into the Unity runtime with WebSocket.

![Imgur](https://i.imgur.com/Opprsx1.png)

## Unity Version

Unity2019.1.12f1 or higher
